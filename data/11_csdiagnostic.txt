CS Diagnostic System Networking. CS Diagnostic System is a tool that transforms your laptop/PC or computer network into an advanced diagnostic system to easily acquire, analyze, store and transport resting ECG, stress tests, ambulatory blood pressure and spirometry measurements. CS Diagnostic System consists of CS Web, CS ECG, CS Stress, CS spirometry, CS ABP and CS Client. CS Diagnostic System can operate in a Citrix** or traditional environment with data security measures to control access and privileges. CS Diagnostic System Networking connects your workstations to a LAN for enhanced access and improved workflow. CS Web enables remote access to reports from virtually anywhere. Multi-level security privileges and unique password assignments support system integrity and protect patient data in CS Diagnostic System Networking. Client floating licenses with auto time-out and log-ins optimize system usage, while increasing the security of your server in CS Diagnostic System Networking. Database synchronization allows off-line or remote device use with auto data merge upon reconnection in CS Diagnostic System Networking.
